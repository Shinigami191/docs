      <p>Setting Dimensions</p>
      <p>CSS can be used to manage dimensions such as visibility, width or height of HTML elements</p>

      <br>

      <p>Properties for Dimensions</p>
      <p>Visibility: Indicates if an element is visible or hidden</p>
      <p>Visible: Default value, the value is visible</p>
      <p>Hidden: The element is hidden (but still takes up space)</p>
      <p>Collapse:Only for table rows, row groups, columns, columns group and flex items that removes a row or column as if display none of used and renders it hidden</p>
      <p>Width: Indicates the width</p>
      <p>Height: Indicates the Height</p>
      <p>Line-height: Indicates the line-spacing</p>
      <p>Max-height: Indicates the maximum height</p>
      <p>Min-height: Indicates the minimum height</p>
      <p>Max-width: Indicates the maximum width</p>
      <p>Min-width: Indicates the minimum width</p>

      <br>

      <p>Pseudo-Classes and Links</p>
   <img src="Pseudo.jpg" alt="Pseudo and Links" class="center-image">
      <p>Defined by a colon and are not user defined</p>
      <p>Specific pseudo-classes can be used on specific HTML elements only</p>
      <p>The anchor element has four pseudo classes</p>
      <p>LVHA (Link, Visited, Hover, Active is the order of pseudo classes)</p>
      <p>The value that you place inside â€¦ will be displayed whenever a pseudo property is performed </p>
      <p>You can set any design in a pseudo class</p>
      <p>Can be used on any element or block element on the web page</p>

      <br>

      <p>Link</p>
      <p>For hyperlinks that have not yet visited</p>
      <p>Set unvisited link characteristics</p>

      <br>

      <p>Visited</p>
      <p>For visited hyperlinks</p>
      <p>Set visited link characteristics or effect</p>

      <br>

      <p>Hover</p>
      <p>For hyperlinks when the mouse is hovered above them</p>
      <p>Set the character of a link element as when the user hovers or moves the mouse over the link</p>
      <p>Should be placed after the link property for it to function properly </p>

      <br>

      <p>Active</p>
      <p>For hyperlinks when they are clicked</p>
      <p>Sets the character of an active link element as when the user clicks the mouse on the link</p>

      <br>

      <p>Focus</p>
      <p>Sets the character of an active link element as when the user clicks and moves the mouse on the link</p>

      <br>
