Lesson 2

Structure of CSS:

- Selector: The HTML element you want to style

- Property: The style attribute you want to modify

- Value: The value you want to apply to the property

- Declaration: A single instruction that defines a style for an element

- Declaration block: Contains one or more declarations for styling rules


Style Sheet:

- A set of instructions to the browser on how to display webpage elements

- Text files (or embedded in HTML) that separate content from design

- Help the browser display pages in a particular way


Statement:

- Identifies elements and tells the browser how to display them


2 Things to Remember:

1. The type attribute should be style type="text/css"

2. CSS code can be enclosed in HTML comments <!--...--> for older browsers


Three Kinds of CSS:

Linked (External):

- Most global type of CSS

- Allows consistent style across pages

- Uses a .css file and is linked via the <link> tag in the <head>


Steps to Create External CSS:

1. Open a blank notepad file

2. Type the styles

3. Save as filename.css with "All Files" as the type

4. Link it in the HTML file using: <link rel="stylesheet" href="style.css" type="text/css">


Embedded (Internal):

- Used for page-specific styles inside the <style> tag in the <head>

- Overrides external CSS


Inline:

- Used for small changes to individual elements

- Applied directly with the style attribute

- Overrides both embedded and external styles


CSS Comments:

- Used to explain the code for readability and maintenance

- Starts with /* and ends with */

- Can span multiple lines
