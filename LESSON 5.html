Lesson 5

CSS Selectors

Used to find or select the HTML elements you want to style

6 types


When to use

If you want to apply the same style to multiple elements, category or purpose

When styling many elements that share a same tyle- even if they have different tags


Element Selector

Selects HTML elements and applies CSS to them


ID Selector

Applied to one element in a page

Normally used ones

Defines by “#” and uses id attribute

Selects the HTML element with a unique identifier (id) and adds CSS to it

Not advisable to start an id selector with a number because some browsers do not support it

Is unique and selects one unique element


When to use

When styling a unique element that appears only once on a page


Universal Selector

Also referred as Wildcard selector

Selects every single HTML element on the page

Written by using an asterisk


When to use

When you want to style all elements on the page


Group Selector

Allows you to select multiple elements and apply the same style to all of them

Uses commas to separate the HTML element


When to use

When different elements share the same styling


Attribute selector

Selects elements based on specific attribute values

Only selects an element if a specified given attribute exist

Syntax is Element{attribute}


When to use

When you want to style elements based on their html attributes


What I have learned:

Understanding different selectors helped me realize how useful l CSS can be. I learned that I can target exactly what I want on the page, whether it’s one unique element or a whole group. This makes my designs more efficient and organized, and I feel more in control when building websites.
