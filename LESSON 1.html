Lesson 1

CSS: Cascading Style Sheets

Describes how HTML elements are to be displayed in media.


Why use CSS?

- Define styles for your web pages

- Provide consistency across web pages

- Saves time

- Improves accessibility and responsiveness

- Enhances design

- Easy maintenance


Capabilities of CSS:

- Easily update your pages

- Position objects

- Layer elements

- Create custom tags


Advantages:

- Saves time and effort

- Faster downloads and page loading

- Better design flexibility

- Easier to maintain and update

- Separation of content and design


Layers of a Webpage:

1. Content Layer – Defines layout and content. The core HTML of the page.

2. Presentation Layer – Controls how users see your page in a browser.

3. Behavior Layer – Adds real-time interaction to your page.


Content Layer Best Practices:

- Use clear, readable text

- Use alt text for media

- Explain abbreviations

- Maintain good contrast

- Notify users of changes


Presentation Layer Best Practices:

- Text zoom must not break layout

- Interactive elements should be easy to find

- Good contrast between text and background

- Clear and consistent navigation


Behavior Layer Best Practices:

- Ensure all functions work regardless of input device

- Make the user experience smooth and easy


What I have learned:

I realized that CSS is way more than just making websites look pretty — it’s what brings structure and life to a webpage. Understanding the three layers (Content, Presentation, Behavior) helped me see how web pages are built from the ground up and how each part affects the user experience.
