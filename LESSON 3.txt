      <h1>Lesson 3</h1>
      <p><strong>Display Property:</strong> Allows you to control how an element is displayed.</p>

      <p><strong>Block Level Element:</strong> Takes up as much space as possible by default and is laid out vertically.</p>

      <p><strong>Inline Level Element:</strong> Displays in a line, does not force text to a new line, and is laid out horizontally. Cannot contain block level elements.</p>

      <p><strong>Importance:</strong> Some CSS properties react differently for each display type.</p>

      <p><strong>Property for Display:</strong></p>
      <p>- <code>display: inline</code>: Inline display or horizontal layout</p>
      <p>- <code>display: block</code>: Block display or vertical layout</p>
      <p>- <code>display: list-item</code>: Display as a list item</p>
      <p>- <code>display: none</code>: Element does not affect layout</p>
      <p>- <code>display: inherit</code>: Inherits from the parent element</p>
    </section>

    <hr>

    <section>
      <h1>Div Tag</h1>
   <img src="Div.jpg" alt="Div" class="center-image">
      <p>Divides the contents of a web page into individual sections.</p>
      <p>Allows styling to specific sections of your webpage.</p>
      <p>Acts as a block-level element and structures your HTML cleanly.</p>
      <p>Should not be used inside a paragraph.</p>
      <p>Creates spacing before and after blocks of content.</p>
    </section>

    <section>
      <h1>Span Tag</h1>
   <img src="Span.jpg" alt="Span" class="center-image">
      <p>Generic inline tag used to emphasize or target specific parts of inline content.</p>
      <p>Often used inside paragraphs or small content chunks.</p>
    </section>

    <section>
      <h1>Span and Div Tags</h1>
      <p>They do not have default styles.</p>
      <p>Need to be styled using attributes like <code>class</code> or <code>id</code>.</p>
      <p>Used with CSS to format parts of content.</p>
    </section>

    <section>

      <p><strong>Class:</strong> Indicates which CSS style will be applied to an element.</p>
